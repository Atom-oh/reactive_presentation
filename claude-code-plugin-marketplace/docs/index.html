Claude Code 플러그인 마켓플레이스를 만들어보자

SA 노하우를 Agent + Skill로 캡슐화하고, 팀 전체의 생산성을 높이는 플러그인 개발기

7 Chapters ~15분 발표 Level 300 Lightning Tech Talk

이 문서에 대하여

이 문서는 Claude Code 위에서 플러그인 마켓플레이스(oh-my-cloud-skills)를 만든 경험을 공유하는 라이트닝 테크톡의 보조 자료입니다. 발표 슬라이드와 함께 읽으면 더 깊이 있는 이해가 가능합니다.

핵심 메시지 AI 에이전트에게는 자유보다 잘 설계된 제약(하네스)이 중요합니다. CLAUDE.md, Hooks, Quality Gate 같은 선언적 메커니즘으로 AI의 행동 범위를 정의하고, 지식을 코드로 축적하여 팀 전체가 활용할 수 있게 만드는 것이 이 프로젝트의 핵심입니다.
슬라이드 보기 이 테크톡의 인터랙티브 슬라이드도 Remarp(이 문서에서 소개하는 프레임워크)로 만들었습니다. Dogfooding의 힘을 직접 확인해 보세요.
