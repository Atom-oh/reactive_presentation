또박(Ttobak) — AI 미팅 어시스턴트

SA 업무에서 미팅이 차지하는 비중은 큽니다. 또박은 회의 녹음부터 AI 요약, 고객 인텔리전스까지 미팅 전체 라이프사이클을 자동화합니다.

핵심 기능

녹음 & 트랜스크립션

  • 마이크/탭 오디오 캡처 (Web Speech API, Chrome getDisplayMedia)
  • AWS Transcribe + Nova Sonic A/B 비교 — 두 STT 엔진의 결과를 나란히 비교
  • 6개 국어 실시간 번역 (Amazon Translate)
  • 오디오 파일 업로드 지원 (m4a, mp3, wav, webm)

AI 분석

  • Bedrock Claude Sonnet으로 미팅 요약 자동 생성
  • 액션 아이템 추출 — 담당자, 우선순위 자동 태깅
  • RAG Q&A — Bedrock Knowledge Base로 과거 미팅 내용 검색/질의

고객 인텔리전스

  • 뉴스 크롤링 — 구독한 고객사의 Google News RSS 자동 수집
  • AI 브리핑 — Sonnet이 SA 관점의 분석을 생성
  • Deep Research — Bedrock Agent가 다중 소스 리서치 수행

아키텍처

CloudFront (ttobak.atomai.click)
  ├─ Lambda@Edge (JWT auth on /api/*)
  ├─ S3 OAC (Next.js static export)
  └─ HTTP API Gateway → ttobak-api Lambda (Go, chi router)

Event-Driven Pipeline:
  audio/ upload → EventBridge → ttobak-transcribe → AWS Transcribe
  transcripts/ → EventBridge → ttobak-summarize → Bedrock Claude → DynamoDB

Crawler Pipeline:
  EventBridge (daily) → Step Functions → TechCrawler + NewsCrawler → S3 KB

MCP Server 연동

또박은 MCP Server를 제공하여 Claude Code에서 직접 미팅 데이터에 접근할 수 있습니다:

  • OAuth PKCE 인증 — 보안 토큰 기반 접근
  • 미팅 데이터 쿼리 — "지난 미팅에서 고객이 언급한 기술 스택은?"
  • 요약/액션 아이템 조회 — 프로그래밍적으로 활용 가능
인프라 9개 CDK 스택으로 완전 자동화 배포. GitHub Actions + 셀프 호스팅 러너로 CI/CD 구성.
